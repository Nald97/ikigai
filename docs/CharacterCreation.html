    CharacterCreation

    new CharacterCreation() → {JSX.Element}

    This component is the main component for the character creation process. It is responsible for rendering the different steps of the character creation process. It also handles the data that is passed from one step to the next. It is also responsible for saving the data to the database. It is also responsible for incrementing the selected counts of the ikigai elements. It is also responsible for redirecting the user to the dashboard after the character creation process is completed. It is also responsible for redirecting the user to the dashboard if the user is already logged in. It is also responsible for redirecting the user to the login page if the user is not logged in. It is also responsible for displaying a toast message if the user is not logged in.
